[Naming Guide]
- 네이밍은 영문(소문자)과 하이픈(-)만을 사용
- ID는 카멜형식(nameId)

타이틀 : h-
배경 : bg-
버튼 : btn-
아이콘 : ico-
섬네일 : thumb-
배너 : bnr-
팝업 : popup-
블릿 : blt-
네비게이션 : gnb-, lnb-, snb-, tab-
라인 : line-
컨텐츠 이미지(텍스트 포함) : p-
링크 : link-area


버튼 네이밍(컴포넌트명-속성-상태)

기본 버튼 스타일(btn-default)
- 버튼텍스트 기본 중앙정렬
- 사이즈: large/medium/small/height/width
- 색상: red,black,gray,blue
- 정렬: left/center/right

btn-text
btn-text-on

btn-large-color
btn-medium-line
btn-small-text
btn-circle-color
btn-circle-line

로그인 : btn-login
검색 : btn-sch / btn-sch-on 
탑 : btn-top
더보기 : btn-more
페이지 :  btn-prev / btn-next
